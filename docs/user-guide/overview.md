# User Guide

Welcome to Balto. This guide explains how to use the app — whether you are a **dog owner** looking to book a walk or a **walker** looking to manage your schedule.

## Roles

Balto has two distinct user roles that share the same login system:

### Dog Owner

A dog owner can:
- Create a profile and add their pets
- Browse walkers in the Services tab
- Book a walk for a specific pet, date, and duration
- Watch their dog's walk live on a map while the walk is active
- See the route, distance, and duration summary after the walk ends
- Cancel upcoming bookings

### Walker

A walker is also a registered user who has applied to become a walker and been approved. A walker can:
- Set up a walker profile (bio, hourly rate, service area)
- Set their weekly availability and block out exception dates
- Receive booking requests and accept or reject them
- Start a walk and have their GPS position broadcast to the owner in real time
- End the walk and see a summary with route, distance, and duration

:::info Can I be both owner and walker?
Yes. The same account can have both owner and walker capabilities. The app shows walker-specific features (Bookings tab, Start Walk button) once the walker profile is approved.
:::

## App navigation

After logging in you see a five-tab bottom navigation bar:

| Tab | Icon | What it does |
|---|---|---|
| **Home** | House | Dashboard — live walk card, upcoming walks, quick links |
| **Walks** | Location pin | Full list of your bookings (in-progress / pending / upcoming / history) |
| **Services** | People | Browse walkers, vets, and pet stores |
| **Coach** | Sparkle | AI chat assistant for dog-related questions |
| **Profile** | Paw | Your profile, pets, settings |
