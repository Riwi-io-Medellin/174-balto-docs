# Live Walk Tracking

When the walker starts your dog's walk, Balto lets you watch the route on a live map in real time.

## How it appears

You don't need to do anything to start tracking — Balto detects the walk automatically.

**On the Home screen**, a green **"Walk in progress"** card appears with the message _"Your dog is out! Tap to see live location."_

**On the Walks tab**, the booking card shows a **"Live"** badge in green.

Tap either to open the live map.

## The live map screen

The live map shows:

| Element | Description |
|---|---|
| **Blue marker** | Walker's current GPS position, updated in real time |
| **Green polyline** | The route taken so far |
| **Green start marker** | Where the walk began |
| **Elapsed time** | Timer counting up from walk start |
| **Distance** | Kilometres walked so far (calculated from GPS points) |

The map camera automatically follows the walker — you don't need to scroll or zoom manually.

## Connecting states

When you open the live map, the screen may show one of these states briefly:

| State | What it means |
|---|---|
| **Connecting to walker…** | The app is establishing a real-time connection to the server |
| **Waiting for walker to start** | The walk hasn't begun yet (you booked, but walker hasn't pressed Start Walk) |
| **Walk in progress** | Connected and receiving position updates |

If you see "Could not connect", tap **Retry**. The app will re-establish the connection.

## Losing and regaining connection

The connection uses automatic reconnect. If your phone briefly loses signal, it recovers silently without showing an error. The route polyline simply resumes from where it left off.

## When the walk ends

When the walker taps End Walk, the live map transitions automatically to the **Walk Summary** screen. You do not need to close and reopen anything.

[Learn about the walk summary →](/user-guide/owner/walk-history)

## Privacy note

The live map is only visible to the owner who booked the walk. The walker's location is not shared with any other users.
